# 🚀 Lead Generation & Reporting System

## 📌 Overview
This project is an automated data pipeline that:
- Scrapes business leads from Justdial
- Cleans and processes data
- Generates Excel reports with insights and charts

## 🛠️ Tech Stack
- Python
- Selenium
- Pandas
- OpenPyXL

## ⚡ Features
- Multi-city lead scraping
- Email extraction from websites
- Data cleaning & deduplication
- Automated Excel report generation
- Summary insights + charts

## 📊 Output
- Raw Leads
- High Quality Leads
- Summary Dashboard

## ▶️ How to Run
```bash
pip install -r requirements.txt
python main.py
